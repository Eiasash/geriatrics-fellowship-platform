üöΩ Urinary Incontinence

Types & Treatment:

  • Stress: Pelvic floor exercises, pessary
  • Urge: Bladder training, Oxybutynin (avoid if cognitive impairment)
  • Overflow: Treat obstruction, intermittent cath
  • Functional: Scheduled toileting, improve access

Reversible Causes (DRIP):

  • Delirium/Dementia
  • Restricted mobility
  • Infection/Inflammation
  • Polyuria (DM, diuretics)
